# RestApi_JWT_Auth_Service

## Описание

Rest Api сервис авторизации позволяет пользователям регистрироваться и аутентифицироваться. Он использует JWT для создания токенов, которые обеспечивают доступ к защищенным ресурсам.

- **Технологии:**
  - Go
  - Docker
  - Chi Router
  - JWT
  - PostgreSQL

- **Эндпоинты:**
  - `POST /register` - регистрация нового пользователя
  - `POST /auth` - аутентификация пользователя
  - `GET /validate` - валидация токена
